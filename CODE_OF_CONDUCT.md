# Code of Conduct

## Our pledge

This project is a curated research atlas. We want contributing, reviewing, and
discussing entries to be a welcoming experience for everyone, regardless of
background, identity, or level of experience.

## Expected behavior

- Be respectful and constructive in issues, pull requests, and reviews.
- Focus feedback on the resource and the catalog, not the person.
- Cite official sources and keep claims about datasets accurate and conservative.
- Assume good faith and help newer contributors follow the inclusion rules.

## Unacceptable behavior

- Harassment, insults, or discriminatory language.
- Personal or political attacks.
- Deliberately adding misleading, inflated, or unverifiable resource claims.
- Publishing others' private information without permission.

## Scope

This Code of Conduct applies within all project spaces — issues, pull requests,
commits, and any other communication tied to this repository.

## Reporting

If you experience or witness unacceptable behavior, please open a private report
by contacting the maintainer through their GitHub profile
([@ChaoYue0307](https://github.com/ChaoYue0307)). All reports will be reviewed
and handled discreetly.
